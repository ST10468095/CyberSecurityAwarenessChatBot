CYBER SECURITY AWARENESS CHAT BOT 

Project descrpiption: The Cyber security awareness chat bot is a console, C# application designed to promote and provide cyber security awareness through the use of an active chat bot. The
user is first greeted by the chat bot through the use of a welcome.wav audio recording. An Ascii interface is then shown to the user describing a "CyberSecurity Chat-bot". The chat bot then 
asks for the users name and then allows the user to ask cyber security related questions to which the chat bot responds with advice/information on the appropriate topic asked by the user.

System Setup: 1: Install Visual Studio
              2: During the installation, slect .NET desktop development workload
              3: Download the project from GitHub
              4: Open the respective file
              5: make sure that the welcome.wav file is present as if it is not present, there will be no audio greeting
              6: Build the solution by pressing Build and tthen build solution 
              7: Run the program by pressing Ctrl + F5

Usage instructions/prompts to put into the chat bot;
password, phishing, browsing, purpose, how are you, virus, antivirus, vpn, wifi, scam, email safety, update, two-factor, identity theft, safe download, backup, ransomware, firewall, hello,
thank.

Current functionality of chat bot: 
Console based chat bot
Cyber security Ascii logo
Audio playback
Captures username + personalized greeting to the user 
Chatbot responses to the user inputs

Status of project: The current stage of the project is in the 1st of three stages where the initial chatbot function has been created that provides basic cyber security information responses.

Future improvements for the project: A more advanced conversation handling with the user, more cyber security topics for the chat bot to speak about with the user aswell as an improved GUI 
interface
